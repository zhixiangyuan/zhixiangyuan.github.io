| 符号      | 含义                                   |
| --------- | -------------------------------------- |
| 0         | 标准输入                               |
| 1         | 标准输出                               |
| 2         | 标准错误输出                           |
| 0<        | 从标准输入中获取                       |
| `1><xxx>` | 将标准输出传输到 `<xxx>`               |
| `2><xxx>` | 将标准错误输出输出到 `<xxx>`           |
| `&><xxx>` | 将标准输出和标准错误输出输出到 `<xxx>` |
| /dev/null | 垃圾桶，丢进去的东西全会消失           |
